# **App Name**: GuanaBet

## Core Features:

- Bet Initiation: Users can define an event, winning/losing outcomes, time limit, and bet amount in satoshis to create a new peer-to-peer bet.
- Lightning Network Deposit: Integrates with the Lightning Network for participants to place their bets securely using satoshis.
- Shareable Bet Links: Generates unique, shareable URLs for newly created bets, allowing easy distribution via social media or direct messaging.
- Anonymous Participation: Users can join bets using their Lightning Network address, select predefined outcomes, or propose new ones.
- Outcome Agreement Phase: After the bet deadline, participants can submit their agreed outcome for resolution.
- AI Referee Assistant Tool: In cases of dispute, a randomly selected referee can use an LLM tool to analyze conflicting participant claims and contextual event data to help determine the official bet outcome.
- Lightning Network Payout: Automates the distribution of collected satoshis equally among all winning participants via Lightning Network.

## Style Guidelines:

- Primary color: A rich, digital violet (#7B33CC) for a modern, trustworthy, and engaging feel. It contrasts well with the dark theme and suggests innovation and depth.
- Background color: A very dark, subtle purplish-grey (#2C2631), providing a sleek, tech-oriented backdrop that enhances readability in a low-light setting.
- Accent color: A bright, energetic sky blue (#33A1FF) used for calls-to-action, highlights, and interactive elements, symbolizing clarity and the electric speed of the Lightning Network.
- All text will use 'Inter', a grotesque-style sans-serif, chosen for its modern, objective, and neutral appearance, ensuring clarity and professionalism suitable for a financial application.
- Clean, minimalistic, and modern icons with clear lines to represent actions, bet statuses, and Lightning Network integration elements.
- A responsive, card-based layout designed for easy navigation on both mobile and desktop, prioritizing the display of active bets and their current status.
- Subtle and quick animations for state changes, payment confirmations, and bet result disclosures, providing clear visual feedback without being distracting.
